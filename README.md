# Sales Analysis Dashboard

## Project Overview
This project analyzes sales performance using Microsoft Excel. The dataset was cleaned, transformed, and visualized to identify key business insights related to revenue, profit, products, regions, and salesperson performance.

## Key Insights

- Q4 generated the highest revenue.
- Laptops contributed the highest sales revenue.
- Earphones delivered the highest profit margin.
- North region achieved the best overall sales performance.
- Top-performing salespersons were identified through dashboard analysis.
